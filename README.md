# VSRegex

[ChinaMobilePhoneNumberRegex] wrappers for iOS and macOS in Objective-C.

## Requirements

- iOS 8.0+ / macOS X 10.9+ / tvOS 9.0+ / watchOS 2.0+

## Quick Start

```objc
#import "VSRegex.h"

BOOL isMatch = [VSRegex matchesWithString:@"+8613800138000"];
NSLog(@"%d", isMatch); // YES
```

## Installation

### CocoaPods

```
pod 'VSRegex-Objc', '~> 1.0.0'
```
